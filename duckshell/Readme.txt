**Welcome to duckshell project!**
What is duckshell? duckshell is project you can custom shell by using bash!
how to enable custom. (C++ Version only!)
- enable custom startup
create file .duckrc on your home folder.
and now write a bash script after you open duckshell that will disabled default startup and show new startup!
- enable custom user@host ~$
create file .duckuserhost and write your script
this exmaple
# WARNING FOR ECHO PLESE USE -n BEAUSE DONT THAT NEWLINE!
echo -n "username@hostname $"
save and enjoy!
